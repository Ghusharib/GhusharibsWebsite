Request for access to C Project

We created an Emulator, Assembler and Quiz App Server on a Raspberry Pi

Emulator for an ARM 11 Processor. This involved taking a binary input file and emualting the effects of the binary input on a virtual register. We created a virtual memory, multiple registers including a program counter and CPSR register, and stepping through each binary line. As we proceed, the control is passed to various helper methods which update the registers as required and processes until the end of the file is reached. This can emulate simple instructions like mov as well as far more complex instructions involving Single Data Transfer, Shifting and more. See photo on left for an example of an emulated file.

Assembler for an ARM 11 Processor. Making use of some of the code from the emulator, the assembler takes an assembly file involving mnemonics such as mov and converts them into binary outputs to be saved to a file. This file can then be interpreted by our Emulator or indeed an actual ARM 11 Processor such as on a Raspberry Pi (which is used to flash an LED). Significant complexities include having two pass in order to store values too large to be held in memory at the end of the file instead.

Raspberry Pi Server for a quiz app We made use of a Raspberry Pi in order to create a web server. This web server makes use of CGI scripts and dynamic HTML code in order to provide a user with an interface. The user can then proceed through the quiz and answer questions being greeteed with a final score at the end. This is as part of the extension to the C Project.
