Hardware Alarm System - Link to Github - Back to Projects





For my hardware coursework, I was tasked with creating an alarm system

The alarm system was intended to unlock and disable the alarm when the correct input was fed. The input being 11110.

This involved creating a finite state machine which progressed through various modes. At each mode, the alarm had the possibility of being triggered or going to the next stage upon which the next correct digit would have to be entered. Once all the correct digits were entered, the alarm would disable and the door would be unlocked.

The first image shows the logic gates and setup created at the end to make the alarm system. It involves three inputs, DATA, CLOCK and RESET. These along with three D-Q flip flops create the alarm system. The circuit is minimised by using as many NAND and NOR gates as possible as these use less space on a physical chip.

The second image shows the output of the generated alarm system. The output shows that upon the correct input, the alarm is turned off and the door is unlocked.

More detail can be seen in the final report on the github repo
