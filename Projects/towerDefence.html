C++ Tower Defence Game - Link to Github - Back to Projects

For my A Level Project, I created a Tower Defence Game written in C++

This game is made completely from scratch and makes use of the very effective SFML Graphics Library available for C++. Making use of royalty free images for the enemies and weapons, this game looks well designed.

In creating the game, I have made use of vectors in order to store details on enemies and had my first experience with Object Orientated Programming. I created a class for enemies and weapons as well as for a game and combined it all into a while loop.

The game progresses as you kill more and more enemies. As more die, your score increases and based on the score, the game gets harder. More and more weapons can be bought as cash increases.

The game is well documented as can be seen inside the git repo, and the development is done through stages including feedback and testing from selected users.
